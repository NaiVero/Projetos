# 💡 Calculadora de consumo elétrico inteligente
- 🔍 O objetivo desta calculadora é ajudar usuários a estimar quanto um aparelho gasta de energia elétrica por mês, com base em dados simples de uso.
# 💻 Sistema da calculadora
- ✏️ Como executar o programa: digite o nome do aparelho, depois, a potência em W e por fim, o tempo médio de uso diário em horas e então receberá o resultado do consumo mensal em kWh.
- 🔣 Fórmula utilizada para calcular o consumo mensal: potência x horas diárias x 30 : 1000
